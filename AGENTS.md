# Automatizaciones n8n

Este repositorio incluye el kit de automatizaciones n8n para Cursor.

Al trabajar con n8n, automatizaciones o workflows, usa la skill **`automatizaciones-n8n`** (`.cursor/skills/automatizaciones-n8n/SKILL.md`).

**Conexión n8n:** credenciales en `.env` (ver `.env.example`). MCP configurado en `.cursor/mcp.json`.

**Regla:** no crear ni modificar workflows en la instancia del usuario sin confirmación explícita.

Instrucciones completas: `kit-automatizaciones-n8n/INSTRUCCIONES.md`.
