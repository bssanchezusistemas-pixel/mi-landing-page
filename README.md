# Flavor House — Restaurant Landing Page

Premium dark-luxury landing page demo for a high-end burger restaurant. Built with vanilla HTML, CSS, and JavaScript.

## Live demo

Deploy to [Vercel](https://vercel.com) — see deployment section below.

## Project structure

```
├── index.html          # Main entry point
├── css/
│   └── style.css       # Styles & animations
├── js/
│   └── main.js         # Scroll effects, parallax, interactions
├── images/             # Food photography
│   ├── hero-burger.png
│   ├── signature-burger.png
│   └── meat-platter.png
├── assets/             # Reserved for favicons, icons, etc.
├── vercel.json         # Vercel static site config
└── README.md
```

## Local preview

Open `index.html` in a browser, or use a local server:

```bash
npx serve .
```

## Deploy to Vercel

1. Push this repository to GitHub.
2. Import the repo at [vercel.com/new](https://vercel.com/new).
3. Framework preset: **Other** (static site).
4. Root directory: `.` (project root).
5. Deploy — no build command required.

## Features

- Fullscreen cinematic hero with parallax
- Glassmorphism menu cards with 3D tilt
- Scroll reveal animations & progress bar
- Testimonials marquee
- Contact form, map embed, WhatsApp button
- Fully responsive (mobile-first)
